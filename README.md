## cat-ui 是基于Taro开发的小程序组件（😝造轮子为了学习和日常使用）

## 使用方式
```
git clone git@github.com:zhangfu-git/cat-ui.git
// 拷贝cat-ui 到 自己的Taro目录的src下面
import { CatButton } from './cat-ui';
```

## 目前拥有的组件列表 （使用文档待完善）
* CatAvtar   头像组件
* CatButton  按钮组件
* CatIcon    图标组件
* catList    普通列表组件
* CatPanel   面板组件
* CatRate    评分组件

这个是积累型UI库, 会随着项目不断的添加组件。
